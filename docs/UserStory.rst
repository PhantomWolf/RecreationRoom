User Stories
============================================================
1.用户注册一个账号，系统产生一个唯一的认证链接，并发送邮件到用户的邮箱，用户点击该链接，完成确认，此后账户就可以登录了。
    - 确认未完成时，用户登录只会收到提示：请完成邮箱确认。
    - 用户可以要求重发确认邮件
2.用户输入用户名与密码登录，密码在用户的本地hash过后，发送给auth模块。模块对密码加上salt后再次hash，与数据库中的结果对比，相同则登录成功。
    - 登录失败时，提示用户名或密码错误，并要求输入验证码。
